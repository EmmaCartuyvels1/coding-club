You need a while-loop. Example: playing dices. Play until you get more than 10 throwing two dices. ```r result <- 0 n <- 0 while (result < 10) { n <- n + 1 # game number dice <- sample(1:6, size = 2, replace = TRUE) result <- sum(dice) print(paste0("Game number: ", n, ". Result: ", result)) } "Game number: 1. Result: 8" "Game number: 2. Result: 3" "Game number: 3. Result: 9" "Game number: 4. Result: 11" ``` Many videogames are actually a big while loop! --- background-image: url({{ site.baseurl}}/assets/images/background_challenge_3.png) class: left, middle # Challenge 3. The Fox slot machine

The goal of a fox is to reproduce before it dies.

The fox dies for sure after 10 years in the wild. But it can die earlier due to poison (one of the possible `foods`) or by an enemy (all `enemies` except `"safe"`).

The "fox game" ends when the fox succeeds to reproduce or when it dies.

By using the code provided in [`20200528/20200528_challenges.R`] and using loop techniques: 1. simulate the fox life. Does the fox succeed to reproduce? Or does it die before? 2. Extend the game to a population of 10 foxes. How many foxes succeeded to reproduce?
